# ✨ Disney Character Generator

## 📖 Project Description

The Disney Character Generator is a responsive web application that fetches and displays random Disney character information using the Disney API. The application is built using HTML, CSS, and JavaScript without any external frameworks or libraries.

🎭 Users can view a random Disney character when the page loads and generate a new character by clicking the button. The app dynamically fetches real-time data from the API and updates the UI instantly.

⚡ This project demonstrates API integration, asynchronous programming using Async/Await, error handling with Try/Catch, and responsive web design principles.

🎨 The interface is clean, modern, and interactive with smooth hover effects and a loading indicator for better user experience.

🖼️ The character image is displayed in a card layout with hover animation and HD-style rendering for better visual quality.
